# Volt2Force

Volt2Force is an instrumentation project that measures the impact force generated by a pneumatic hammer using a 2.5klb load cell. The system converts the electrical signal from the load cell into calibrated force measurements, allowing analysis of strike magnitude and repeatability.

## Overview

A compression load cell is placed in the hammer’s impact path. When the hammer strikes, the sensor produces a small voltage proportional to the applied force. This signal is amplified and recorded, then converted into force values through calibration.

The system enables basic force vs. time measurements for pneumatic hammer strikes.

## Experimental Apparatus

- Compression 2.5klb load cell

- Analog signal amplifier

- DAQ for data acquisition

- Mechanical mounting fixture for impact testing

## Workflow

1. Hammer strike applies force to the load cell

2. Load cell produces a small electrical signal

3. Signal is amplified and processed by a DAQ

4. Python script displays data sampling in real-time

5. Calibration converts voltage readings into force values

## Applications

- Impact force measurement

- Pneumatic tool testing

- Dynamic load experimentation
